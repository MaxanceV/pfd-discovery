# PFD-Discovery

[cite_start]Ce projet porte sur la découverte de **dépendances fonctionnelles basées sur des motifs (Pattern Functional Dependencies - PFDs)** pour l'amélioration de la qualité des données[cite: 1, 6]. [cite_start]L'objectif est de comparer une approche algorithmique classique avec une approche assistée par une **IA agentique**[cite: 300, 321].

## Objectifs du Projet
[cite_start]Le projet s'articule autour de trois axes principaux[cite: 533]:
1. [cite_start]**Approche Classique :** Implémentation d'un algorithme de découverte de PFDs (extraction de motifs, génération de candidats et validation)[cite: 535].
2. [cite_start]**Workflow Agentique :** Mise en œuvre d'une solution utilisant des LLMs pour guider la recherche ou enrichir les données[cite: 537].
3. [cite_start]**Évaluation :** Comparaison des performances et de la qualité des règles découvertes sur des jeux de données réels[cite: 540, 560].

## Technologies
* **Langage :** Python
* [cite_start]**IA :** Utilisation de LLMs (Mistral, Claude, etc.) pour la partie agentique[cite: 575].
